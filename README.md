# Agentic LLM

A minimal Next.js (App Router) + Tailwind CSS chat that simulates an LLM without external API keys. Deployable to Vercel.

## Development

```bash
pnpm i # or npm i / yarn
pnpm dev
```

## Build

```bash
pnpm build
```

## Deploy

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-2b367084
```
